---

:::important

This guide applies to Windows **only**. Country flag emojis are supported on other operating systems by default.

:::

Windows does not have country flag emojis by default. This will cause country tabs in Subway Builder to render Unicode flag characters instead of the intended flag emojis. To solve this, we can download and install a font that supports country flag emojis.

<img
  width="1000"
  alt="image"
  src="/images/docs/railyard/v0.2/country-flag-unicode.png"
/>

## Installation {#installation}
Download the latest version of the Country Flag Emojis for Windows font [here](https://github.com/Chasmical/flag-emojis-for-windows/releases/latest/download/Segoe.UI.Emoji.with.Twemoji.Flags.ttf). Once it is downloaded, open the `.ttf` file and click "Install" to install the font on your system.

Upon completion, restart your PC. When you relaunch Subway Builder, you should now see country flag emojis instead of the Unicode characters.
